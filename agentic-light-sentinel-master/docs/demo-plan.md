# 3–5 min Demo Plan

1) **Open Dashboard**: show the map, legend, slider (2013→2025).
2) **Click Region**: side panel shows radiance + hotspots; trend chart animates.
3) **Trigger Ingest**: run `pnpm ingest` → refresh → 2025 appears.
4) **Detect Hotspots**: run `pnpm hotspots` → `AlertsPanel` updates.
5) **Notify**: run `pnpm notify` → explain email alerting.
6) **Feedback**: submit a note in the side panel → "learn" loop.
7) **Close**: outline path to connect real VIIRS tiles + district drilldown.
